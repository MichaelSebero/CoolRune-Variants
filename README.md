## How to install CoolRune-AMD
. Install Artix s6 XFCE on your computer.

. Run these commands.

```
su
curl -LO https://raw.githubusercontent.com/MichaelSebero/CoolRune-AMD/master/CoolRune-Install.sh
sh CoolRune-Install.sh
```
